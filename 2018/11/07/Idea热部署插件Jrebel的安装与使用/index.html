JRebel 能做什么？

JRebel 是一款热部署插件。当你的 java-web 项目在 Tomcat 中 run/debug 的时候 ,修改某一个 java 文件时，tomcat 并不能将此文件的修改 实时编译 并反映到运行的项目中去。JRebel 就可以解决在项目运行状态run/debug 状态下任意修改 java 文件并动态反馈到运行的项目中。

JRebel 插件 获取与安装

在Idea中在线获取JRebel插件并安装。打开Setting(Ctrl+Alt+S)-Plugins,搜索JRebel。Install 后重启。


JRebel 插件 破解

打开Setting(Ctrl+Alt+S)-JRebel,点击Activate now。

点击I already have a license ,输入激活地址和邮箱，勾选Agree,完成激活。

(参考链接：Jrebel最新激活破解方式(持续更新))

打开Setting(Ctrl+Alt+S)-JRebel,点击Work offline 关闭联网检测。

自此JRebel 激活完成。
